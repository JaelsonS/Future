# Future Laboratório - Site Institucional

Site desenvolvido para o Future Laboratório de Palmeira dos Índios, AL, com foco em responsividade, acessibilidade e SEO. O projeto utiliza HTML5 semântico, CSS3 com variáveis e JavaScript puro para interatividade.

## Características
- Design baseado no estilo da Clínica Rui Faria com cores vermelho vinho e branco.
- Botão flutuante de WhatsApp para agendamento direto.
- Estrutura otimizada para SEO local.
- Totalmente acessível e responsivo.

## Tecnologias
- HTML5
- CSS3 (Flexbox, Grid)
- JavaScript (Vanilla)
- Google Fonts, Font Awesome

## Decisões de design
- Paleta vinho e branco para transmitir confiança e sofisticação na área da saúde.
- Tipografia Exo para títulos (impacto) e Roboto para leitura confortável.
- Layout mobile-first com cartões para destacar serviços e equipe.
